# A Sample Layouts Stream Deck Plugin

![Screenshot](screenshot.png)

This SDK-example shows how to use:
- layouts (built-in and custom)
- setFeedbackLayout
- setFeedback
- load layout from file
- load icons from file
- create icons using svg
